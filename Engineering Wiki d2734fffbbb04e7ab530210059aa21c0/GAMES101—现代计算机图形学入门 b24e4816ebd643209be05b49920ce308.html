投影矩阵和观察矩阵怎么表示？

投影矩阵和观察矩阵是两个常用的矩阵，用于实现3D图形变换。

投影矩阵用于实现透视投影，它将3D坐标变换到2D屏幕空间，具体实现就是将模型坐标系中的坐标进行透视除法，从而实现透视投影效果。

观察矩阵用于实现观察变换，它将3D坐标变换到观察坐标系，它可以用来控制物体的位置，旋转，缩放等。

一般来说，投影矩阵和观察矩阵的具体实现由库来提供，我们可以通过API来设置投影参数和观察参数。

例如在OpenGL中，可以使用glm库来创建投影矩阵和观察矩阵，具体如下：

这两个矩阵的参数怎么确定？

投影矩阵和观察矩阵的参数取决于渲染的场景和需求。

在透视投影矩阵中，通常需要指定视场角（field of view）、宽高比（aspect ratio）、近裁剪面（near clipping plane）和远裁剪面（far clipping plane）。视场角表示观察者能够看到的视野角度，通常取45度或者60度，宽高比表示屏幕的宽度和高度的比值，近裁剪面和远裁剪面表示观察者能够看到的距离范围。

在观察矩阵中，通常需要指定观察点（eye point）、观察目标（look at point）和上方向（up direction）。观察点表示观察者的位置，观察目标表示观察者看向的点，上方向表示屏幕顶部的方向。

这些参数可以根据具体需求来确定,

解释一下Mipmap

Mipmap是一种图像纹理缩放技术。它通过预先生成图像的不同缩放版本来提高图像在3D场景中的渲染效率。在使用Mipmap时，图形引擎会选择最合适的缩放版本来渲染，这样可以减少不必要的图像采样和过滤，提高渲染速度。

那Ripmap呢

Ripmap是一种比Mipmap更高级的图像纹理缩放技术。它预先生成每个像素的多个缩放版本，而不是整个图像的缩放版本。这样做可以更好地保留图像的细节，特别是在高级3D场景中，可以提高图像的渲染质量。

Ripmap 比 Mipmap 更加耗费资源，因为需要预先生成更多的缩放版本，但是有更高的画质。

Trilinear Interpolation呢

Trilinear interpolation是一种图像插值技术，用于在纹理缩放时进行颜色和纹理坐标插值。

它工作在两个维度上：水平维度和垂直维度。在水平维度上，它在两个相邻的Mipmap级别之间进行线性插值，在垂直维度上，它在纹理坐标上进行线性插值。这样做可以在纹理缩放时保留更多的细节，并减少纹理缩放带来的失真。

Trilinear interpolation 比 Bilinear interpolation 更耗费资源，但是有更高的画质。

解释一下： 在两个相邻的Mipmap级别之间进行线性插值，在垂直维度上，它在纹理坐标上进行线性插值

在两个相邻的Mipmap级别之间进行线性插值，指的是在不同的Mipmap级别之间进行颜色插值。当场景中物体距离相机较远时，需要使用较低分辨率的Mipmap级别来渲染，而当物体距离相机较近时，需要使用较高分辨率的Mipmap级别来渲染。

在垂直维度上，它在纹理坐标上进行线性插值，指的是在纹理坐标上进行颜色插值。当纹理被缩放到不同的尺寸时，纹理坐标会发生变化，Trilinear interpolation 技术会在纹理坐标变化时进行颜色插值，以保证纹理质量。

这样的线性插值可以更好地保留图像的细节，并减少纹理缩放带来的失真。

那Anisotropic Filtering呢

Anisotropic Filtering (AF) 是一种图像过滤技术，用于在纹理缩放时减少纹理失真。它通过改善纹理采样的方向来提高图像质量。

当物体在3D场景中被观察时，纹理可能会被放大，从而导致纹理失真。Anisotropic Filtering 技术可以通过在纹理采样时考虑物体与相机之间的方向来减少纹理失真。这样可以在纹理缩放时更好地保留图像的细节，并使图像看起来更加真实。

Anisotropic Filtering 比 Trilinear interpolation 更耗费资源，但是有更高的画质。

那EWA Filtering呢

EWA Filtering (Elliptical Weighted Average Filtering) 是一种比 Anisotropic Filtering 更高级的图像过滤技术。它通过更精细地考虑纹理采样的方向来提高图像质量。

EWA Filtering 技术基于一种名为椭圆重心法的算法，该算法可以根据纹理采样方向和缩放比例来计算每个采样点的权重。这样可以更好地考虑纹理的方向和比例，从而更好地保留图像的细节。

EWA Filtering 比 Anisotropic Filtering 更耗费资源，但是有更高的画质。

// 构造MVP中的P 
// eye_fov 视野的大小
// aspect_ratio 视野的长宽比
// zNear 最近处的坐标
// zFar 最远处的坐标
Eigen::Matrix4f get_projection_matrix(float eye_fov, float aspect_ratio, float zNear, float zFar) {
    Eigen::Matrix4f projection = Eigen::Matrix4f::Identity();
        
    // 进行透视投影转化为正交投影的矩阵
    Eigen::Matrix4f P2O = Eigen::Matrix4f::Identity();//将透视投影转换为正交投影的矩阵
    P2O<<
        zNear, 0, 0, 0, 
        0, zNear, 0, 0, 
        0, 0, zNear+zFar,(-1)*zFar*zNear, 
        0, 0, 1, 0;

    float halfEyeAngelRadian = eye_fov/2.0/180.0*MY_PI; // 上下视野角的一半
    float t = zNear*std::tan(halfEyeAngelRadian);//top y轴的最高点
    float r=t*aspect_ratio; //right x轴的最大值
    float l=(-1)*r; //left x轴最小值
    float b=(-1)*t; //bottom y轴的最大值
    
    /*
     *
     * 正交投影过程
     *
     */
    //进行一定的缩放使之成为一个标准的长度为2的正方体
    Eigen::Matrix4f ortho1=Eigen::Matrix4f::Identity();
    ortho1 << 
        2 / (r - l), 0, 0, 0, 
        0, 2 / (t - b), 0, 0, 
        0, 0, 2 / (zNear - zFar), 0, 
        0, 0, 0, 1;
    // 把一个长方体的中心移动到原点
    Eigen::Matrix4f ortho2 = Eigen::Matrix4f::Identity();
    ortho2 << 
        1, 0, 0, (-1) * (r + l) / 2, 
        0, 1, 0, (-1) * (t + b) / 2, 
        0, 0, 1, (-1) * (zNear + zFar) / 2, 
        0, 0, 0, 1;
    // 用于正交投影的矩阵
    Eigen::Matrix4f Matrix_ortho = ortho1 * ortho2;
    
    projection = Matrix_ortho * P2O;
    return projection;
}

Eigen::Matrix4f get_rotation(Vector3f axis, float angle)
{
    Eigen::Matrix4f model = Eigen::Matrix4f::Identity();
    Eigen::Matrix3f temp = Eigen::Matrix3f::Identity();
    float ag = angle/180*MY_PI;
    Eigen::Matrix3f tr;
    Eigen::Matrix3f mul;
    mul << 0, -axis[2], axis[1],
        axis[2], 0, -axis[0],
        -axis[1], axis[0], 0;
    tr = cos(ag)*temp + (1-cos(ag))*axis*axis.adjoint() + mul*sin(ag);
    model << tr(0,0), tr(0,1), tr(0,2), 0,
            tr(1,0), tr(1,1), tr(1,2), 0,
            tr(2,0), tr(2,1), tr(2,2), 0,
            0, 0, 0, 1;
    return model;
}

透视校正插值

透视矫正差值（Perspective Correction Residual）指的是在进行透视矫正时，因为图像的失真、畸变等原因，在矫正后图像中可能会存在一些残留的误差，这些误差被称为透视矫正差值。

透视矫正是指将摄影或者计算机生成的图像中的透视变形进行校正，使得图像中的直线或者平面在投影到二维平面上时能够保持平行或垂直关系。在进行透视矫正时，会对图像进行一系列变换，如旋转、缩放、平移等操作，以达到校正的目的。但是，由于图像本身的复杂性，这些变换可能会引入一些误差，从而导致矫正后图像中出现残留误差。

透视矫正差值可以通过各种方法进行减少或者消除，例如使用更高级别的算法、增加校正点的数量或者改变校正点的位置等。在某些应用中，透视矫正差值可以对图像识别和分析等任务产生负面影响，因此需要尽可能减小其影响。
